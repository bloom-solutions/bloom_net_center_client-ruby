# BloomNetCenterClient

Ruby wrapper for BloomNet Center's API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bloom_net_center_client'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bloom_net_center_client
